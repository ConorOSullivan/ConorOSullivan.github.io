Mapping instead of looping in python

Fri 20 June 2014
By Conor O'Sullivan

Python speed increases

Found this great set of tips online and I thought I'd share.

Instead of doing a loop where you call a function each time like this:

a = [] for elt in list: a.append(int(elt))

Use a map() instead to speed up python computations. map() allows the computer to find the function once and then apply it to all elements of the list instead of finding it over and over.

a = map(int, list)
